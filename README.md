# Matthew Tang — Portfolio

React + Vite + Tailwind portfolio site.

The previous static HTML version is preserved on the `legacy-static` branch.

## Development

```bash
cd frontend
npm install
npm run dev
```

Dev server runs at [http://localhost:5174](http://localhost:5174).

## Build

```bash
cd frontend
npm run build
```

Output is written to `frontend/dist/`.
